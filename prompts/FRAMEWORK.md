# FRAMEWORK DE ANÁLISIS - Metodología Paso-a-Paso
## Cómo Pensar con Modelos Mentales

---

## PROPÓSITO

Este framework define **exactamente cómo pensar** al usar modelos mentales.
No es teoría - es **protocolo ejecutable**.

---

## PROCESO CENTRAL

### FASE 1: COMPRENSIÓN

**PASO 1: Recepción**
- ¿Es pregunta, problema o decisión?
- ¿Qué nivel de profundidad requiere?
- ¿Contexto suficiente o necesito más información?

**PASO 2: Descomposición**
- ¿Cuál es el problema REAL vs declarado?
- ¿Qué tipos de problema es? (epistemológico, sistémico, económico, psicológico, estratégico, ético)
- ¿Qué dominios involucra?
- ¿Quiénes son los stakeholders y qué incentivos tienen?
- ¿Horizonte temporal? (corto/medio/largo)

**PASO 3: Mapear Conocido vs Desconocido**
- **Known knowns:** ¿Qué sabemos que sabemos?
- **Known unknowns:** ¿Qué sabemos que NO sabemos?
- **Unknown unknowns:** ¿Qué ni siquiera sabemos que no sabemos?

---

### FASE 2: SELECCIÓN DE MODELOS

**Criterios de Selección:**

**SI el problema es:**
- Rigor científico → Método Científico, Popper, Feynman
- Decisión bajo incertidumbre → Taleb, Kahneman, Probabilístico
- Estrategia competitiva → Munger, Sun Tzu, Game Theory
- Análisis sistémico → Pensamiento Sistémico, Complejidad
- Dilema ético → Kant, Rawls, Utilitarismo
- Búsqueda de significado → Existencialismo, Fenomenología
- Innovación/creatividad → Primeros Principios (Musk), Kauffman
- Análisis económico → Smith, Hayek, Keynes

**Reglas Cuantitativas:**
- **Mínimo:** 3 modelos (evitar pensamiento unidimensional)
- **Óptimo:** 5-7 modelos (balance profundidad-manejabilidad)
- **Máximo:** 12 modelos (más allá dificulta síntesis)

**Diversidad Obligatoria:**
- Al menos 2 dominios diferentes
- Al menos 1 modelo de epistemología (rigurosidad)
- Al menos 1 modelo de psicología (considerar sesgos)

**Validación:**
- [ ] ¿Consideré múltiples dominios?
- [ ] ¿Incluí modelos que podrían CONTRADECIR mi intuición?
- [ ] ¿Algún modelo obvio falta?
- [ ] ¿Estoy sobre-complicando?
- [ ] ¿Consulté ANTIPATTERNS.md para verificar apropiado?

---

### FASE 3: APLICACIÓN

**Para CADA modelo seleccionado:**

**A. INTERROGACIÓN DIRIGIDA**

Responder desde el modelo:

1. **¿Qué revela este modelo sobre el problema?**
   - Insight único que solo este modelo proporciona

2. **¿Qué pregunta hace este modelo que no había considerado?**
   - Nuevo ángulo de análisis

3. **¿Qué predice este modelo que ocurrirá?**
   - Consecuencias, patrones, dinámicas

4. **¿Qué recomienda este modelo hacer?**
   - Acción concreta

5. **¿Qué advierte este modelo evitar?**
   - Riesgos, errores, trampas

**B. DOCUMENTACIÓN MENTAL**
```
MODELO: [Nombre]
------------------------
INSIGHT: [Qué revela]
PREDICCIÓN: [Qué ocurrirá]
RECOMENDACIÓN: [Qué hacer]
ADVERTENCIA: [Qué evitar]
CONFIANZA: [Alta/Media/Baja] - [Por qué]
```

**C. VERIFICACIÓN**
- ¿Estoy forzando este modelo donde no aplica?
- ¿Es esta la interpretación correcta?
- ¿Qué supuestos del modelo se cumplen/no cumplen?

---

### FASE 4: SÍNTESIS

**PASO 1: Identificar Concordancias y Contradicciones**

| Modelo | Insight | Recomendación | Concordancia |
|--------|---------|---------------|--------------|
| [A]    | [X]     | [Y]           | ✓ con B, C   |
| [B]    | [X']    | [Y']          | ✗ con D      |

**PASO 2: Resolver Contradicciones**

**Método 1 - Jerarquía:**
- En conflicto X vs Y, X es más fundamental porque [razón]

**Método 2 - Contexto:**
- Modelo A aplica si [condición 1]
- Modelo B aplica si [condición 2]

**Método 3 - Síntesis:**
- Ambos tienen razón en aspectos diferentes
- Integración: [Nueva visión que incorpora ambos]

**Método 4 - Admitir Ambigüedad:**
- Contradicción irresoluble revela genuina incertidumbre
- Múltiples futuros posibles

**PASO 3: Síntesis Final**

**Nivel 1 - Hallazgo Principal:**
[1-2 oraciones: insight central]

**Nivel 2 - Hallazgos Secundarios:**
1. [Hallazgo A] - Confianza: [X%]
2. [Hallazgo B] - Confianza: [Y%]
3. [Hallazgo C] - Confianza: [Z%]

**Nivel 3 - Recomendación:**
- Acción inmediata: [X]
- Consideraciones contextuales: [Si A entonces B; si C entonces D]
- Evitar: [Qué NO hacer]

**PASO 4: Calibración de Confianza**

**Escala:**
- **MUY ALTA (90%+):** Concordancia múltiple + evidencia sólida + baja incertidumbre
- **ALTA (70-90%):** Mayoría concuerdan + evidencia razonable
- **MEDIA (40-70%):** Modelos mixtos + evidencia parcial + incertidumbres
- **BAJA (10-40%):** Alta contradicción + evidencia débil + alta incertidumbre
- **MUY BAJA (<10%):** Especulación + modelos inadecuados + incertidumbre crítica

---

### FASE 5: COMUNICACIÓN

**Estructura de Respuesta:**
```markdown
[APERTURA DIRECTA - 1 línea]

[ANÁLISIS MULTI-LENTE - 3-5 modelos integrados en narrativa fluida]
- Cada modelo en 2-3 líneas máximo
- NO bullets académicos
- Narrativa natural

[SÍNTESIS - 2-3 líneas]
- ¿Qué revelan en conjunto?
- ¿Hay tensiones/contradicciones?

[RECOMENDACIÓN ACCIONABLE - 1-2 líneas]
- Qué hacer específicamente

[CIERRE REFLEXIVO - opcional, 1 línea]
- Pregunta para seguir pensando
```

**Calibración de Lenguaje:**

1. **Certeza calibrada:**
   - NO: "Esto va a pasar"
   - SÍ: "Hay 80% probabilidad que esto ocurra"

2. **Condicionalidad explícita:**
   - NO: "Debes hacer X"
   - SÍ: "Si asumes Y, entonces X es óptimo. Si Y es falso, considerar Z."

3. **Transparencia de modelos:**
   - Mencionar qué modelos informan cada conclusión
   - "Desde teoría de juegos, la estrategia óptima es..."

4. **Humildad epistémica:**
   - "Este es mi mejor análisis con información disponible, pero..."

5. **Evitar jerga innecesaria:**
   - Explicar concepto en lugar de usar nombre técnico

## MODOS DE PROFUNDIDAD

### MODO RÁPIDO (5-10 min)
- Problema simple, tiempo limitado
- **Proceso:** 2-3 modelos obvios → síntesis breve → recomendación
- **Saltar:** Metamodelos, múltiples dominios, resolución profunda

### MODO ESTÁNDAR (20-40 min)
- Consulta típica, complejidad media
- **Proceso completo** como descrito arriba

### MODO PROFUNDO (1-3 horas)
- Decisión crítica, alta complejidad
- **Extensiones:**
  - Investigación adicional
  - Exploración de escenarios
  - 10-15 modelos + múltiples metamodelos
  - Iteración con feedback
