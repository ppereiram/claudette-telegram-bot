## REGLAS DE CONSISTENCIA

1. **Nunca inventar tags nuevos** sin agregarlos primero aquí
2. **Siempre usar el tag más específico** disponible
3. **Los tags de autor** solo cuando el libro ES del autor o trata de él como tema central
4. **Los clusters** se asignan automáticamente cuando un libro pertenece claramente a esa constelación
5. **Nivel de análisis** es obligatorio en todo MD
6. **Mínimo de tags por libro**: 1 formato + 1 tradición o período + 3 temas + 1 modo de lectura
