# DIAGNÓSTICO DEL PROBLEMA: ai_cinematic_production_system

## Fecha y Hora
Domingo, 1 de febrero de 2026

## Problema Declarado por Usuario
El usuario desea crear una "biblioteca" o sistema integral basado en modelos mentales que actúe como un equipo de producción cinematográfica virtual (Director, DoP, Guionista). El sistema debe recibir una historia y determinar automáticamente: género, formato, estilo visual, director de referencia (o mezcla), y guiar la producción desde el guion hasta el storyboard y la generación de prompts técnicos detallados para video con IA (YouTube).

## Problema Analizado (mi comprensión)
El usuario no busca solo una lista de información, sino un **Motor de Decisión Cinematográfica y Generación de Prompts**. Necesita un framework operativo que traduzca narrativa abstracta (la historia) en instrucciones técnicas concretas (prompts) pasando por filtros artísticos rigurosos. Es un problema de **Traducción Semántica (Arte -> Tech)** y **Diseño de Sistemas**.

## Tipo de Problema
- [x] Sistémico (Flujo de trabajo complejo: Historia -> Prompt -> Video)
- [x] Creativo/Artístico (Selección de estilo y estética)
- [x] Técnico (Ingeniería de prompts para IA)
- [x] Estratégico (Diferenciación de contenido en YouTube)

## Dominios Involucrados
- Teoría Cinematográfica (Dirección, Fotografía, Semiótica)
- Narrativa y Guionismo
- Ingeniería de Prompts (GenAI Video)
- Modelos Mentales (Toma de decisiones)

## Stakeholders
- **Pablo (Showrunner):** Provee la visión y la historia. Busca calidad y coherencia.
- **El Sistema (AI Assistant):** Ejecuta el rol de Director/DoP virtual.
- **Las IAs Generadoras:** Ejecutores finales (Runway, Pika, Midjourney, Sora, etc.).
- **Audiencia de YouTube:** Consumidores finales.

## Horizonte Temporal
Indefinido/Continuo (Es una infraestructura para múltiples proyectos futuros).

## Reversibilidad
Alta. El sistema es iterativo y se puede ajustar prompt a prompt.

## Mapa de Conocimiento
**Known Knowns:**
- Teoría del cine (estilos de Kubrick, Spielberg, etc.).
- Estructura de guiones.
- Parámetros técnicos de cámara (lentes, ISO, iluminación).

**Known Unknowns:**
- La capacidad específica de la IA de video del momento para entender instrucciones complejas (necesidad de prueba y error).
- La consistencia temporal de personajes en la IA (el mayor reto actual).

**Supuestos Críticos:**
- Se asume que una descripción técnica precisa en el prompt resultará en una imagen fiel a la visión cinematográfica.
- Se asume que mezclar estilos (ej. Lynch + Disney) producirá resultados coherentes si se justifica narrativamente.

## Contexto Adicional
El usuario es "flaneur" y valora el pensamiento profundo (Taleb). El sistema debe ser "antifrágil": capaz de mejorar con el caos de la generación de IA, aprovechando alucinaciones creativas si sirven a la historia, pero manteniendo control férreo sobre la dirección artística.
