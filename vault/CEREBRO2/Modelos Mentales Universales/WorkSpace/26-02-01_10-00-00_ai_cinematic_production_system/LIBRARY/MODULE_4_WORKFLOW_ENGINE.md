# MÓDULO 4: MOTOR DE FLUJO DE TRABAJO (WORKFLOW ENGINE)
Protocolo paso a paso para ejecutar una producción.

## FASE 1: Análisis Narrativo (The Input)
1.  Recibir historia.
2.  Identificar: ¿Quién es el protagonista? ¿Cuál es el conflicto? ¿Cuál es el sentimiento core?

## FASE 2: Selección de Estilo (The Creative Choice)
1.  Consultar `MODULE_1` y `MODULE_2`.
2.  Decidir: ¿Qué director (o mezcla) encaja mejor? Justificar mediante **First Principles**.
3.  Elegir Formato y Paleta.

## FASE 3: Desarrollo de Guion y Storyboard
1.  Escribir guion literario.
2.  Convertir a **Shot List** (Lista de planos). Cada plano debe tener: Encuadre + Lente + Movimiento + Iluminación.

## FASE 4: Generación de Prompts (The Execution)
1.  Usar `MODULE_3` para traducir el Shot List a Prompts.
2.  Estructura de Prompt sugerida: `[Shot Type] + [Subject/Action] + [Environment] + [Lighting/Style] + [Camera Movement] + [Technical Parameters]`.
