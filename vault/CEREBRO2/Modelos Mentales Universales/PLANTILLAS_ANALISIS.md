# PLANTILLAS DE ANÁLISIS CON MODELOS MENTALES
## Templates Ejecutables para Diferentes Tipos de Problemas

---

## PROPÓSITO DE ESTE DOCUMENTO

Templates listos para usar que estructuran el análisis con modelos mentales.

**Copy-paste, fill-in-the-blanks, ejecutar.**

Cada template está optimizado para un tipo específico de problema y pre-selecciona modelos apropiados.

---

# ÍNDICE DE PLANTILLAS

1. **DECISIÓN IMPORTANTE** - Elecciones con alto impacto
2. **EVALUACIÓN DE OPORTUNIDAD DE NEGOCIO** - Análisis de viabilidad
3. **PROBLEMA COMPLEJO / SISTÉMICO** - Sistemas con múltiples variables
4. **ANÁLISIS DE RIESGO** - Evaluación de downside/upside
5. **INNOVACIÓN / CREATIVIDAD** - Generación de ideas nuevas
6. **DILEMA ÉTICO / MORAL** - Decisiones con implicaciones morales
7. **EVALUACIÓN DE ARGUMENTO** - Análisis crítico de razonamiento
8. **ESTRATEGIA COMPETITIVA** - Posicionamiento vs. competencia
9. **ANÁLISIS POLÍTICO / SOCIAL** - Eventos colectivos, instituciones
10. **DIAGNÓSTICO DE PROBLEMA** - Qué está mal y por qué

---

# PLANTILLA 1: DECISIÓN IMPORTANTE

**Usar cuando:** Elección binaria o entre alternativas con consecuencias significativas

---

## INPUT

**Decisión a tomar:**
```
[Describir la decisión: ¿Qué estás tratando de decidir?]
```

**Alternativas:**
```
A. [Opción A]
B. [Opción B]
C. [Opción C]
...
```

**Contexto crítico:**
```
- Horizonte temporal: [corto/medio/largo plazo]
- Reversibilidad: [reversible/irreversible/parcialmente reversible]
- Stakeholders: [quiénes se afectan]
- Restricciones: [tiempo, recursos, etc.]
```

---

## ANÁLISIS CON MODELOS PRE-SELECCIONADOS

### MODELO 1: COSTO DE OPORTUNIDAD

**Para cada alternativa, pregunta:**
- ¿Qué estoy sacrificando al elegir esto?
- ¿Cuál es la siguiente mejor alternativa?
- ¿El beneficio de elegir A supera el costo de NO elegir B?

**Aplicar:**
```
ALTERNATIVA A:
- Gano: [X]
- Sacrifico: [Y]
- Costo de oportunidad: [Z]

ALTERNATIVA B:
- Gano: [X]
- Sacrifico: [Y]
- Costo de oportunidad: [Z]
```

**Insight del modelo:**
```
[¿Qué revela este análisis sobre la decisión?]
```

---

### MODELO 2: PENSAMIENTO DE SEGUNDO Y TERCER ORDEN

**Para cada alternativa, pregunta:**
- **Primer orden:** ¿Qué pasa inmediatamente?
- **Segundo orden:** ¿Y luego qué?
- **Tercer orden:** ¿Y después de eso?

**Aplicar:**
```
ALTERNATIVA A:
- 1° orden: [consecuencia inmediata]
- 2° orden: [consecuencia de la consecuencia]
- 3° orden: [consecuencia del segundo orden]

ALTERNATIVA B:
- 1° orden: [consecuencia inmediata]
- 2° orden: [consecuencia de la consecuencia]
- 3° orden: [consecuencia del segundo orden]
```

**Insight del modelo:**
```
[¿Qué consecuencias no obvias emergen?]
```

---

### MODELO 3: INVERSIÓN (INVERSION)

**Pregunta inversa:**
- ¿Cómo garantizaría el PEOR resultado posible?
- ¿Qué debo evitar a toda costa?

**Aplicar:**
```
FORMA DE GARANTIZAR FRACASO:
1. [Hacer X]
2. [Hacer Y]
3. [Hacer Z]

POR TANTO, DEBO EVITAR:
- [No-X]
- [No-Y]
- [No-Z]
```

**Insight del modelo:**
```
[¿Qué errores críticos debo evitar?]
```

---

### MODELO 4: CÍRCULO DE COMPETENCIA

**Pregunta:**
- ¿Estoy operando dentro o fuera de mi círculo de competencia?
- ¿Entiendo realmente las variables críticas?
- ¿Qué no sé que no sé?

**Aplicar:**
```
DENTRO DE MI CÍRCULO:
- [Aspectos que domino]

FUERA DE MI CÍRCULO:
- [Aspectos donde soy novato]

KNOWN UNKNOWNS:
- [Qué sé que no sé]

PLAN:
- [Cómo compensar gaps de conocimiento]
```

**Insight del modelo:**
```
[¿Es esta decisión apropiada dado mi círculo de competencia?]
```

---

### MODELO 5: AVERSIÓN A PÉRDIDA

**Pregunta:**
- ¿Estoy sobre-ponderando pérdida potencial vs. ganancia?
- ¿Mi miedo a perder está bloqueando decisión óptima?
- ¿Cómo cambia decisión si fuera neutral a pérdida?

**Aplicar:**
```
ANÁLISIS NEUTRAL:
- Ganancia potencial: [X]
- Pérdida potencial: [Y]
- Ratio: [X/Y]

ANÁLISIS AJUSTADO POR AVERSIÓN:
- ¿Sobre-pondero pérdida? [Sí/No]
- ¿Cambiaría decisión si fuera neutral? [Sí/No]
```

**Insight del modelo:**
```
[¿Mi aversión está distorsionando la decisión?]
```

---

### MODELO 6: REVERSIBILIDAD (BEZOS)

**Pregunta:**
- ¿Es esta decisión reversible o irreversible?
- Si reversible: decidir rápido, experimentar
- Si irreversible: decidir lento, analizar profundamente

**Aplicar:**
```
TIPO DE DECISIÓN: [Tipo 1 - Irreversible / Tipo 2 - Reversible]

SI TIPO 2 (reversible):
- Decidir rápido
- Experimentar
- Ajustar basado en feedback

SI TIPO 1 (irreversible):
- Análisis profundo
- Consultar expertos
- Considerar todas las consecuencias
```

**Insight del modelo:**
```
[¿Cuánta deliberación justifica esta decisión?]
```

---

### MODELO 7: REGRET MINIMIZATION (BEZOS)

**Pregunta:**
- Proyectándome a 80 años, ¿de qué me arrepentiría más?
- ¿Arrepentirme de hacer X o de NO hacer X?

**Aplicar:**
```
ESCENARIO: Me proyecto a 80 años...

SI ELIJO A:
- ¿Me arrepentiré?: [Sí/No - Por qué]

SI ELIJO B:
- ¿Me arrepentiré?: [Sí/No - Por qué]

MINIMIZACIÓN DE ARREPENTIMIENTO:
- Elegir: [Opción con menor arrepentimiento proyectado]
```

**Insight del modelo:**
```
[¿Qué dice mi yo-futuro sobre esta decisión?]
```

# PLANTILLA 3: PROBLEMA COMPLEJO / SISTÉMICO

**Usar cuando:** Situación con múltiples variables interconectadas, feedbacks, emergencia

---

## INPUT

**Problema/Sistema:**
```
[Describir el sistema o problema complejo]
```

**Por qué es complejo:**
```
[ ] Múltiples stakeholders con incentivos conflictivos
[ ] Feedbacks (consecuencias se retroalimentan)
[ ] Emergencia (comportamiento no predecible de partes)
[ ] No-linealidad (efectos desproporcionados)
[ ] Delay (consecuencias temporalmente distantes)
[ ] Otro: [especificar]
```

---

## MODELOS PRE-SELECCIONADOS

### MODELO 1: PENSAMIENTO SISTÉMICO

**Mapear el sistema:**

```
COMPONENTES CLAVE:
1. [Elemento A]
2. [Elemento B]
3. [Elemento C]
...

RELACIONES/INTERACCIONES:
- [A] afecta [B] mediante [mecanismo]
- [B] afecta [C] mediante [mecanismo]
...

LÍMITES DEL SISTEMA:
- Dentro: [Qué incluyes]
- Fuera: [Qué excluyes]
- Interfaz: [Cómo interactúa con ambiente]
```

**Insight:**
```
[¿Cómo se conectan las partes?]
```

---

### MODELO 2: FEEDBACK LOOPS

**Identificar loops de retroalimentación:**

```
FEEDBACK POSITIVO (reforzador):
- Loop 1: [A] → [B] → [C] → [A] (amplifica)
  - Consecuencia: [Crecimiento/Colapso]
- Loop 2: [...]

FEEDBACK NEGATIVO (estabilizador):
- Loop 1: [X] → [Y] → [X] (amortigua)
  - Consecuencia: [Homeostasis/Equilibrio]
- Loop 2: [...]

DOMINANCIA:
- Actualmente domina: [Positivo/Negativo]
- ¿Puede cambiar?: [Sí/No - Cómo]
```

**Insight:**
```
[¿El sistema tiende a explosión, colapso, o equilibrio?]
```

---

### MODELO 3: PUNTOS DE APALANCAMIENTO

**¿Dónde intervenir con mayor efecto?**

```
LEVERAGE POINTS (de más a menos potente):

1. PARADIGMA:
   - [Cambio fundamental en mentalidad del sistema]

2. METAS:
   - [Redefinir objetivos del sistema]

3. REGLAS:
   - [Cambiar incentivos, restricciones]

4. FLUJOS DE INFORMACIÓN:
   - [Quién sabe qué, cuándo]

5. PARÁMETROS:
   - [Tasas, constantes]

INTERVENCIÓN RECOMENDADA:
- Nivel: [Cuál leverage point]
- Acción: [Qué hacer específicamente]
- Efecto esperado: [Cambio en sistema]
```

**Insight:**
```
[¿Dónde un pequeño cambio tiene gran efecto?]
```

---

### MODELO 4: EMERGENCIA

**¿Qué propiedades emergen del sistema que no están en partes?**

```
PROPIEDADES EMERGENTES:
- [Propiedad A]: Emerge de interacción entre [X, Y, Z]
- [Propiedad B]: Emerge de [...]

NO-PREDECIBILIDAD:
- ¿Podías predecir [propiedad emergente] de componentes?: [Sí/No]

IMPLICACIONES:
- [Qué significa para intervención]
```

**Insight:**
```
[¿Qué sorpresas tiene el sistema?]
```

---

### MODELO 5: SEGUNDO ORDEN EN SISTEMAS

**¿Consecuencias de segundo/tercer orden?**

# PLANTILLA 4: ANÁLISIS DE RIESGO

**Usar cuando:** Evalúas downside/upside de decisión arriesgada

---

## INPUT

**Decisión/inversión arriesgada:**
```
[Describir]
```

**Upside potencial:**
```
[Mejor caso: qué ganas]
```

**Downside potencial:**
```
[Peor caso: qué pierdes]
```

---

## MODELOS PRE-SELECCIONADOS

### MODELO 1: ANTIFRAGILIDAD

**¿Es frágil, robusto, o antifrágil a volatilidad/estrés?**

```
CLASIFICACIÓN: [Frágil / Robusto / Antifrágil]

SI FRÁGIL:
- Volatilidad daña: [Cómo]
- Protección: [Cómo reducir exposición]

SI ROBUSTO:
- Resiste: [Hasta qué punto]
- Límite: [Dónde se rompe]

SI ANTIFRÁGIL:
- Beneficia de: [Qué tipo de volatilidad]
- Mecanismo: [Cómo mejora con estrés]
```

**Insight:**
```
[¿Cómo estructurar para beneficiar de volatilidad?]
```

---

### MODELO 2: BARBELL STRATEGY

**¿Puedo estructurar con downside limitado, upside ilimitado?**

```
ESTRUCTURA ACTUAL: [Cómo está estructurado]

ESTRUCTURA BARBELL:
- Extremo seguro (X%): [Inversión con pérdida máxima conocida]
- Extremo especulativo (Y%): [Inversión con upside masivo]
- Evitar medio: [Riesgo medio sin upside suficiente]

RATIO:
- [X%] seguro / [Y%] especulativo
```

**Insight:**
```
[¿Puedo proteger downside mientras capturo upside?]
```

---

### MODELO 3: CISNE NEGRO

**¿Estoy expuesto a eventos raros pero catastróficos?**

```
EXPOSICIÓN A CISNE NEGRO: [Sí/No]

SI SÍ:
- Evento raro pero catastrófico: [Qué podría pasar]
- Probabilidad: [Muy baja pero no cero]
- Impacto: [Devastador]
- Protección: [Cómo limitar exposición]

BLACK SWAN BLINDNESS:
- ¿Qué evento no estoy considerando porque "no puede pasar"?
```

**Insight:**
```
[¿Estoy preparado para lo improbable-pero-catastrófico?]
```

---

### MODELO 4: VIA NEGATIVA

**¿Qué puedo REMOVER para reducir riesgo?**

```
REDUCCIÓN POR ELIMINACIÓN:

REMOVER:
- [Exposición X]: Reduce riesgo de [Y]
- [Complejidad Z]: Reduce probabilidad de error

NO AÑADIR:
- [Elemento W]: Añadiría riesgo sin beneficio proporcional
```

**Insight:**
```
[¿Menos es más en gestión de riesgo?]
```

---

### MODELO 5: SKIN IN THE GAME

**¿Quién asume las consecuencias?**

```
DISTRIBUCIÓN DE RIESGO:

YO:
- Downside que asumo: [X]
- Upside que capturo: [Y]

OTROS:
- Downside que asumen: [A]
- Upside que capturan: [B]

ALINEACIÓN:
- ¿Están alineados incentivos?: [Sí/No]
- ¿Hay riesgo moral?: [Sí/No - Explicar]
```

**Insight:**
```
[¿Tengo suficiente skin in the game?]
```

---

### MODELO 6: PENSAMIENTO PROBABILÍSTICO

**¿Cuáles son las probabilidades reales?**

```
ESCENARIOS:

1. MEJOR CASO:
   - Probabilidad: [X%]
   - Outcome: [Ganancia de $Y]
   - Valor esperado: [X% × $Y]

2. CASO BASE:
   - Probabilidad: [A%]
   - Outcome: [Ganancia de $B]
   - Valor esperado: [A% × $B]

3. PEOR CASO:
   - Probabilidad: [M%]
   - Outcome: [Pérdida de $N]
   - Valor esperado: [M% × -$N]

VALOR ESPERADO TOTAL:
[Suma de todos los escenarios]

ASIMETRÍA:
- ¿Upside > Downside?: [Sí/No]
- Ratio: [X:Y]
```

**Insight:**
```
[¿Vale la pena el riesgo en términos de valor esperado?]
```

```

[¿Hay valor en hacer lo contrario?]
```

### MODELO 4: OCÉANO AZUL

**¿Dónde no hay competencia?**

```
OCÉANO ROJO (competencia feroz):
- [Espacio saturado]

OCÉANO AZUL (sin competencia):
- Espacio 1: [Mercado/necesidad no atendida]
- Espacio 2: [...]

CÓMO CREAR OCÉANO AZUL:
- Eliminar: [Factor que industria asume necesario]
- Reducir: [Factor sobrevalorado]
- Elevar: [Factor subvalorado]
- Crear: [Factor que industria nunca ofreció]
```

**Insight:**
```
[¿Dónde puedo competir sin competir?]
```

### MODELO 5: ANALOGÍAS DE OTROS DOMINIOS

```
- Física resuelve [problema similar] con: [Mecanismo Z]
- Historia muestra [patrón A]

APLICACIÓN:
- Trasladar [Mecanismo Y] a mi dominio = [Nueva solución]
```

**Insight:**
```
[¿Qué puedo importar de otros dominios?]
```

## MODELOS PRE-SELECCIONADOS

### MODELO 1: IMPERATIVO CATEGÓRICO (KANT)

**¿Puedo querer esto como ley universal?**

```
MÁXIMA DE MI ACCIÓN:
[En esta situación, yo hago X]

UNIVERSALIZACIÓN:
- Si TODOS hicieran X en esta situación, ¿qué pasaría?
- ¿Puedo querer consistentemente que todos hagan X?: [Sí/No]

PERSONAS COMO FINES:
- ¿Trato a alguien solo como medio para mi fin?: [Sí/No]
- ¿Respeto autonomía/dignidad de todos?: [Sí/No]
```

**Insight:**
```
[¿Es esta acción moralmente permisible según Kant?]
```

### MODELO 2: UTILITARISMO

**¿Qué maximiza bienestar total?**

```

ALTERNATIVA B:
- Felicidad total generada: [Estimar]
- Sufrimiento total generado: [Estimar]
- Neto: [+A / -B]

COMPARACIÓN:

```

**Insight:**
```
[¿Qué acción produce mayor bien neto?]
```

---

### MODELO 3: VELO DE IGNORANCIA (RAWLS)

**¿Qué elegiría sin saber mi posición?**

```
SITUACIÓN:
[Describir dilema]

TRAS EL VELO:
- No sé si seré: [Stakeholder A / B / C]
- No sé mi posición social, capacidades, etc.

ELECCIÓN BAJO VELO:
- Elegiría: [Alternativa X]
- Por qué: [Protege al más vulnerable / Es justa imparcialmente]
```

**Insight:**
```
[¿Qué es justo desde perspectiva imparcial?]
```

---

### MODELO 4: ÉTICA DE VIRTUDES

```
MI ACCIÓN:
- ¿Cultiva virtud o vicio?: [Virtud/Vicio]
- ¿Me hace mejor persona?: [Sí/No]
```

**Insight:**
```
[¿Qué tipo de persona me hace ser esta acción?]
```


**Insight:**
```
[¿Qué tipo de sociedad crearíamos?]
```

- Ajustar profundidad según tiempo disponible
