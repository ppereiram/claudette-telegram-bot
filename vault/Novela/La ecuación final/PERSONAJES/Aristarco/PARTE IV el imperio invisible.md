4.1 PROYECTO JANO: LA CARA COMO MERCANCÍA
Fecha de inicio: Marzo 2018
Nombre código: Jano (el dios romano de dos caras—apropiado)
Inspiración: Clearview AI + tecnología de reconocimiento facial masivo
Objetivo: Unificar todas las bases de datos de biometría facial del mundo en un solo sistema
CONTEXTO TECNOLÓGICO:
En 2018, Pedro identificó una oportunidad que otros consideraban "éticamente problemática"—él la llamó "inevitable y lucrativa."
El mercado global de biometría facial estaba fragmentado:
Clearview AI escaneaba redes sociales (40,000M de imágenes)
Gobiernos construían bases de datos de ciudadanos (pasaportes, DNI)
Aeropuertos fotografiaban viajeros (obligatorio para "seguridad")
Smartphones desbloqueaban con Face ID (conveniente, invasivo)
Cámaras urbanas capturaban transeúntes (vigilancia "inteligente")
Hospitales escaneaban pacientes (verificación de identidad médica)
El problema: Todos capturaban. Nadie integraba.
La solución de Pedro: V.A.L.I.S. haría lo que nadie más podía: unificar todas esas bases de datos fragmentadas en una sola.
No mediante hackeo (demasiado ilegal).
Mediante adquisiciones corporativas opacas y contratos de "interoperabilidad."

"El Cliente retiene propiedad de todos los datos biométricos capturados. El Proveedor [subsidiaria de Aureus] procesa dichos datos exclusivamente para propósitos de verificación de identidad del Cliente según términos de este contrato.
El Proveedor puede utilizar datos agregados y anonimizados, derivados de múltiples clientes, para mejorar precisión de algoritmos de detección facial, siempre y cuando ningún dato individual identificable sea compartido entre clientes o utilizado para propósitos comerciales no relacionados con servicios de seguridad."
